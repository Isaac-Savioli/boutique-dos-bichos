# 🐾 Boutique dos Bichos

Site institucional de um petshop fictício, desenvolvido para fins acadêmicos no 1º semestre da Fatec.

## 🌐 Acesse o projeto
👉 [Clique aqui para visualizar o site](https://isaac-savioli.github.io/boutique-dos-bichos/)

---

## 📌 Sobre o projeto

O **Boutique dos Bichos** é um site criado com o objetivo de simular a presença digital de um petshop, apresentando seus serviços, estrutura e diferenciais de forma moderna e responsiva.

---

## ✨ Funcionalidades

- Página inicial com apresentação do petshop
- Seção de serviços (banho e tosa, veterinário, day care)
- Seção “Sobre nós”
- Área com pets (clientes)
- Seção “antes e depois”
- Layout responsivo (adaptado para mobile)

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## 📱 Responsividade

O site foi desenvolvido para funcionar em:
- Desktop  
- Tablets  
- Dispositivos móveis  

---

## 🎨 Design

O layout foi prototipado no Figma, seguindo uma identidade visual moderna com foco em usabilidade e experiência do usuário.

---

## 📚 Objetivo

Este projeto tem como objetivo praticar:
- Estruturação de páginas web  
- Estilização com CSS  
- Responsividade  
- Organização de projeto front-end
